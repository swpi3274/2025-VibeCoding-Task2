個人品牌 · 前端工程師

這裡是個人網站 Hero 區的占位內容，將來會放入自我介紹與主視覺文案。

目前這一區只是骨架，用來說明未來會放入的一句話自我介紹、核心職稱（例如 Frontend Developer）、以及呼叫行動按鈕（例如「瀏覽作品集」或「下載履歷」）。實際文案與版面會在之後用 Section SDD 的 Prompt 產生，再整段貼回來取代這裡。

給開發者的提示：之後會用 Section SDD 的 Prompt 產生新的 section，請把新的 <section id="hero" …> 程式碼整段貼在 「<!-- SECTION:HERO-START -->」與「<!-- SECTION:HERO-END -->」這兩個註解中間，完全取代現在這一區。

你可以在新的設計中加入：視覺 Banner、主要標語、角色標籤（如「前端工程師、RWD 網站、UI 實作」）、以及導向作品集或聯絡方式的按鈕。

ABOUT

關於我（占位內容，之後會用 Section SDD 替換）

這裡是 About 區的基礎骨架，用來放個人背景、自我介紹與職涯輪廓。實際內容、排版與細部結構預計在下一階段透過 Section SDD 的 Prompt 產生，再整段覆蓋這個區塊。

給開發者的說明：之後會用 Section SDD 的 Prompt 產生新的 <section id="about" class="section section-about">，請把新的 section 程式碼整段貼在 「<!-- SECTION:ABOUT-START -->」與「<!-- SECTION:ABOUT-END -->」這兩個註解中間，完全取代現在這一區。

可以預期未來會在這裡放入：一到兩段個人簡介、履歷概述（例如年資、主要領域、常用技術）、以及簡短的職涯目標敘述。若有時間線、里程碑或個人成就，也可以在之後的版本中加入更精細的結構。

提醒：無論將來怎麼改版，建議維持 .section-about 與 .section-title 這類 class，不僅方便覆寫 CSS，也能讓其他 Prompt 更容易針對這一區調整樣式。

文字區佔位：可以想像這裡是「精簡版履歷」，例如：

  • — 目前職稱與主要負責的工作內容。
  • — 過去在前端開發或相關領域的經驗摘要。
  • — 對使用者體驗、程式品質或團隊協作的核心價值觀。

視覺或補充內容佔位：

  • — 未來可以改成列表或小卡片展示「關鍵里程碑」。
  • — 也可以嵌入另一張橫式形象照片或軟實力標籤。
  • — 保留這個欄位，方便之後改成更多欄位排版。
SKILLS

技能區（占位內容，未來會補上完整技能矩陣）

這裡是 Skills 區的基本骨架，用來放前端技術、相關工具與軟實力。實際的技能列表、熟悉程度標示與分類方式，都預計在下一階段用 Section SDD 精細定義，再將整個 section 取代。

替換方式說明：之後會用 Section SDD 的 Prompt 產生新的 <section id="skills" class="section section-skills">，請把新的 section 程式碼完整貼在 「<!-- SECTION:SKILLS-START -->」與「<!-- SECTION:SKILLS-END -->」之間，整段覆蓋現在的內容。

建議保留 .section-skills、.skills-grid 等 class 名稱，讓未來只要覆寫對應的 CSS Slot，就能快速切換版面風格。

預計會放的內容類型示意：

前端基礎 HTML / CSS / RWD 語意化標記 跨瀏覽器相容

也可能包含：

進階技能 元件化思維 設計系統實作 效能與可維護性

之後可在 Section SDD 中定義技能顯示方式（例如條列、標籤雲、等級條或分組卡片），這裡只先預留「技術分組」與「標籤」的概念，方便演化。

PROJECTS

作品集（占位內容，未來會放實際專案卡片）

這裡是 Projects 區的佔位骨架，預期會放上 2～6 個代表性作品，包括個人 side project、練習作品、或實際商業專案。每張卡片可以包含專案名稱、簡介、使用技術與連結。

替換說明：未來若使用 Section SDD 產生新的作品集區塊，請將 <section id="projects" class="section section-projects"> 的完整程式碼，貼在 「<!-- SECTION:PROJECTS-START -->」與「<!-- SECTION:PROJECTS-END -->」之間，完全取代這一區。

建議保留 .project-card 與 .projects-grid 這類 class 名稱，有利未來在 CSS:RWD 區塊中擴充更複雜的網格版面或 hover 效果。
